Why?

My series of cheatsheets have led me through the design process of laying out and positioning elements on the page, and how best to select the elements to apply styling.

Now that I have my building blocks in place and my background set it is time to look at a very simple but possibly the most important type of content on any project....the text.

All text inside an element is all affected as one single entity (only <span>, <strong>, and similar, or pseudo-elements (::first-letter) can be used to style subsections of text).

Font Size

Size is commonly set as px, em, rem, vw, ch or pt (print view). 16px is the usual browser default.

p {
font-size: 1em;
}

It can also be set using the values xx-small, x-small, small, medium, large, x-large or xx-large, as well as relatively using smaller and larger

  • H1

  • H2

  • H3

  • H4

  • H5
  • H6
  • P

To make font-size responsive we can use clamp()...

p {
font-size: clamp(1rem, 2.5vw, 2rem);
}

For this example 1rem is the smallest size we want tthe font be, 2.5vw is the ideal and 2rem is the largest size.

Text Drop Shadows

}
  1. The horizontal offset of the shadow from the original text
  2. The vertical offset of the shadow from the original text
  3. The blur radius (default = 0)
  4. Base colour (default = black)

Multiple shadows can be added...

p {
text-shadow:
4px 4px 5px red,
2px 2px 3px black,
3px 3px 4px green;
}

Line Height

Line Height as the name suggests sets the height of each line with line-height using most size and length values and a unitless value (recommend 1.5-2)

p {
line-height: 1.5;
}

Letter Spacing

As the name suggests sets the space between letters with letter-spacing using most size and length values and is used more for heading text than in blocks of text.

p {
letter-spacing: 4px;
}

Word Spacing

Also as the name suggests sets the space between words with word-spacing using most size and length values.

p {
word-spacing: 6px;
}

Font Variant

Switch between 'Normal' and Small Caps.

p {
font-variant: small-caps;
}

Font Kerning

This sets the use of the kerning information stored in a font

  • auto
  • normal
  • none
p {
font-kerning: auto;
}

Text Indent

Sets how much horizontal space should be left before the beginning of the first line with text-indent using most size and length values.

p {
text-indent: 4em;
}

White Space

As it sounds this is used to set how white space is handled

  • normal
  • nowrap
  • pre
  • pre-wrap
  • pre-line
  • break-spaces
p {
white-space: pre;
}

Text Overflow

This sets how hidden overflow content is signaled to users. The container also needs to have overflow and white-space set before text-overflow will take affect.

Values can be:

  • clip
  • ellipsis
  • 'custom string'
p {
white-space: nowrap;
overflow: hidden;
text-overflow: "----";
}

Word Break

Using word-break specifies how words should break when reaching the end of a line.

  • normal
  • keep-all
  • break-all
p {
word-break: break-all;
}

Word Wrap

Using word-wrap can allow long words to be able to break and wrap onto the next line.

  • normal
  • break-word
p {
word-wrap: break-word;
}

Writing Mode

This sets whether lines of text are laid out horizontally or vertically, as well as the direction in which blocks progress.

  • horizontal-tb
  • vertical-rl
  • vertical-lr
p {
writing-mode: vertical-rl;
}

Text Orientation

This only affects text in vertical mode and text-orientation can change the orientation (for example if a verticle table header was needed

  • mixed
  • upright
  • sideways-right
  • sideways
p {
writing-mode: vertical-rl;
text-orientation: sideways-right;
}

Font

Multiple properties can also be set through the shorthand property font.

In this order:font:font-style, font-variant, font-weight, font-stretch, font-size/line-height, font-family;

p {
font:italic normal bold normal 3em/1.5 Helvetica, Arial, sans-serif;
}

Only font-size and font-family are required when using font shorthand
