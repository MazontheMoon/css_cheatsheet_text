Why?

My series of cheatsheets have led me through the design process of laying out and positioning elements on the page, and how best to select the elements to apply styling.

Now that I have my building blocks in place and my background set it is time to look at a very simple but possibly the most important type of content on any project....the text.

All text inside an element is all affected as one single entity (only <span>, <strong>, and similar, or pseudo-elements (::first-letter) can be used to style subsections of text).

Font Size

Size is commonly set as px, em, rem, vw, ch or pt (print view). 16px is the usual browser default.

p {
font-size: 1em;
}
  • H1
  • H2
  • H3
  • H4
  • H5
  • H6
  • P

To make font-size responsive we can use clamp()...

p {
font-size: clamp(1rem, 2.5vw, 2rem);
}
